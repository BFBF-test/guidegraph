À propos de Guidegraph

Guidegraph est un outil libre pour créer des infographies institutionnelles conformes aux bonnes pratiques : lisibilité, contrastes accessibles, hiérarchie visuelle, formats adaptés au contexte.

Il vous guide en 5 étapes : définir votre objectif, choisir le format adapté, construire votre charte graphique, remplir les blocs de contenu, puis vérifier et exporter.

Entièrement hors ligne. Guidegraph ne nécessite aucune connexion internet. Aucune donnée ne quitte votre ordinateur — tout reste dans votre navigateur.

Sauvegarde automatique. Votre travail est sauvegardé dans le navigateur à chaque modification (icône disquette). Vous pouvez aussi télécharger un fichier de sauvegarde (.json) et le recharger à tout moment.

Version v1 (preuve de concept) — 6 étapes opérationnelles : objectif, format, mise en page, charte graphique, contenu, export (PNG, SVG, HTML, PDF).

Installer une police sur mon ordinateur

Une fois installée, votre police sera disponible dans tous vos logiciels. Pour l'utiliser dans Guidegraph, importez directement le fichier .ttf ou .otf dans l'outil.

Windows
Mac
Linux
  1. Téléchargez le fichier .ttf ou .otf sur votre ordinateur
  2. Clic droit sur le fichier → Installer
  3. La police est immédiatement disponible dans vos logiciels
  4. Dans Guidegraph : cliquez sur "Importer ma propre police" et sélectionnez le fichier
  1. Téléchargez le fichier .ttf ou .otf
  2. Double-cliquez sur le fichier → Installer la police
  3. Ou ouvrez Livre des polices et glissez-déposez le fichier
  4. Dans Guidegraph : cliquez sur "Importer ma propre police" et sélectionnez le fichier
  1. Créez le dossier ~/.local/share/fonts/ si nécessaire
  2. Copiez le fichier .ttf ou .otf dans ce dossier
  3. Exécutez fc-cache -f -v dans le terminal
  4. Dans Guidegraph : cliquez sur "Importer ma propre police" et sélectionnez le fichier
